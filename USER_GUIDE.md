# TourManager Admin Panel - User Guide

## Getting Started

### Logging In

1. **Navigate to the login page**
   - Open the application in your browser
   - You'll automatically be directed to the login page

2. **Enter your credentials**
   
   **For Super Admin access:**
   - Email: `admin@tourmanager.com`
   - Password: `admin123`
   
   **For Staff access:**
   - Email: `staff@tourmanager.com`
   - Password: `staff123`

3. **Click "Login"**
   - You'll be redirected to your personalized dashboard

### Understanding Your Dashboard

The dashboard you see depends on your role:

**Super Admin Dashboard shows:**
- Comprehensive metrics (Total Groups, Travellers, Tours, Messages, Documents)
- Charts showing trends and analytics
- Date range filters for custom reporting

**Staff Dashboard shows:**
- Your active tours
- Upcoming tours
- Total travellers assigned to you
- Today's messages
- Pending invites

## Navigation

### Sidebar Menu

The left sidebar contains all main navigation options:

- **Dashboard** - Home page with overview statistics
- **Groups** - Manage all tour groups
- **Staffs** - (Super Admin only) Manage team members
- **Individual Chats** - Private messages with travellers
- **Broadcasts** - Send announcements to groups
- **Documents** - Manage all tour documents
- **Reports** - (Super Admin only) Analytics and exports
- **Settings** - (Super Admin only) System configuration
- **Logout** - Sign out of the system

### Collapsing the Sidebar

Click the arrow button at the top of the sidebar to collapse/expand it for more screen space.

## Core Features

### Managing Tour Groups

#### Creating a New Group

1. Navigate to **Groups** from the sidebar
2. Click **"Create Group"** button
3. Fill in the details:
   - Group Name (e.g., "European Adventure 2026")
   - Description
   - Start Date
   - End Date
   - Enable/Disable Group Chat
   - Enable/Disable Broadcast
4. Click **"Create Group"**

#### Editing a Group

1. Find the group in the list
2. Click the **Edit** (pencil) icon
3. Modify the details
4. Click **"Update Group"**

#### Viewing Group Details

1. Click the **View** (eye) icon on any group card
2. You'll see tabs for:
   - **Travellers** - Manage group members
   - **Group Chat** - View and send messages
   - **Private Chats** - One-on-one conversations
   - **Documents** - Group-specific files
   - **Settings** - Group configuration

### Managing Travellers

#### Adding Travellers

1. Open a group's detail page
2. Go to the **Travellers** tab
3. Click **"Add Traveller"**
4. Enter:
   - Name
   - Mobile Number
5. Click **"Add Traveller"**

#### Sharing Invite Links

1. In the Travellers tab, you'll see a unique invite link
2. Options:
   - **Copy** - Copy link to clipboard
   - **WhatsApp** - Share directly via WhatsApp
   - **SMS** - Send via text message

### Using Chat Features

#### Group Chat

1. Open a group's detail page
2. Go to the **Group Chat** tab
3. View all messages from staff and travellers
4. To send a message:
   - Type in the message box at the bottom
   - Press **Enter** or click **Send**
5. To attach files:
   - Click the attachment icon
   - Select your file (images, PDFs, documents)

#### Private Chats

1. Navigate to **Individual Chats** from the sidebar
2. Select a traveller from the left panel
3. Chat window opens on the right
4. Send messages and attachments just like group chat

### Broadcasting Messages

#### Sending a Broadcast

1. Navigate to **Broadcasts** from the sidebar
2. Click **"New Broadcast"**
3. Select the tour group
4. Type your message
5. Preview the message
6. Click **"Send Broadcast"**

All travellers in the selected group will receive the message.

#### Viewing Broadcast History

The Broadcasts page shows all previously sent messages with:
- Date and time sent
- Recipient count
- Message content
- Sender name

### Managing Documents

#### Uploading Documents

1. Navigate to **Documents** from the sidebar
2. Click **"Upload Document"**
3. Select the tour group
4. Either:
   - Drag and drop files into the upload area
   - Click to browse and select files
5. Click **"Upload"**

Supported file types: PDF, Word, Excel, Images (up to 10MB)

#### Downloading Documents

1. Find the document in the grid
2. Click the **Download** icon
3. File will download to your device

#### Deleting Documents

1. Find the document in the grid
2. Click the **Delete** (trash) icon
3. Confirm the deletion

## Super Admin Features

### Managing Staff

#### Adding New Staff

1. Navigate to **Staffs** from the sidebar
2. Click **"Add Staff"**
3. Fill in:
   - Name
   - Mobile Number
   - Email Address
   - Role (Staff or Super Admin)
4. Configure permissions
5. Click **"Add Staff"**

#### Editing Staff

1. Find the staff member in the table
2. Click the **Edit** icon
3. Modify details
4. Click **"Update Staff"**

#### Activating/Deactivating Staff

Toggle the switch in the Status column to activate or deactivate a staff member.

### Viewing Reports

1. Navigate to **Reports** from the sidebar
2. Set date range filters:
   - Start Date
   - End Date
3. View metrics and charts:
   - Tours created over time
   - Travellers per tour
   - Invite acceptance rates
   - Chat activity
4. Export options:
   - Click **"Export as PDF"** for PDF report
   - Click **"Export as Excel"** for spreadsheet

### Configuring Settings

1. Navigate to **Settings** from the sidebar
2. Configure sections:

**App Branding:**
- Change application name
- Upload custom logo

**File Upload Settings:**
- Set maximum file size
- Define allowed file types

**Notification Preferences:**
- Toggle email notifications
- Toggle SMS notifications
- Toggle push notifications

**Default Tour Settings:**
- Enable/disable chat by default
- Enable/disable broadcasts by default
- Auto-archive completed tours
- Require invite approval

3. Click **"Save Settings"** when done

## Tips & Best Practices

### Organization

- **Use descriptive group names** - Include destination and year
- **Add detailed descriptions** - Help travellers understand the tour
- **Keep documents organized** - Use clear file names
- **Review pending invites regularly** - Follow up with non-responders

### Communication

- **Use broadcasts for announcements** - Weather updates, schedule changes
- **Use group chat for discussions** - Let travellers interact
- **Use private chats for personal matters** - Sensitive information
- **Respond promptly** - Maintain traveller satisfaction

### Document Management

- **Upload key documents early** - Visa info, itineraries, emergency contacts
- **Keep files up to date** - Remove outdated information
- **Use appropriate file formats** - PDFs for read-only, Word for editable
- **Organize by group** - Easy to find relevant documents

### Security

- **Log out when done** - Especially on shared computers
- **Don't share credentials** - Each staff member should have their own account
- **Review staff access regularly** - Deactivate former employees
- **Back up important data** - Export reports periodically

## Troubleshooting

### Can't Log In?

- Verify your email/mobile and password
- Check for typos
- Contact your administrator if locked out

### Don't See Expected Menu Items?

- Menu visibility depends on your role
- Contact Super Admin if you need additional permissions

### Document Upload Failed?

- Check file size (must be under 10MB)
- Verify file type is allowed
- Try a different browser
- Check your internet connection

### Messages Not Sending?

- Verify you have chat permissions
- Check your internet connection
- Try refreshing the page
- Contact technical support if issue persists
